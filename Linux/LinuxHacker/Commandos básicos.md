
##### El $PATH
El PATH sirve para que puedas ejecutar commandos sin necesidad de poner la ruta absoluta de este mismo. Por ejemplo:
- Quiero utilizar el commando `whoami`, ese commando se encuentra en la ruta `/usr/bin`, por lo tanto, el PATH existirán diferentes rutas separadas por un : que vas a permitirte ejecutar un commando sin necesidad de colocar la ruta absoluta
>En ves de colocar en la terminal `/usr/bin/whoami`
>`$PATH=/usr/bin`
>Ya con esa configuración yo puedo simplemente ejecutar `whoami`

##### which
Sirve para conocer la ruta absoluta o el alias de algún commando
>También se puede remplazar por `command -v`

##### ls
Funciona para ver el contenido de archivos de algún directorio.
Con -l puedes ver los permisos de los archivos [[Lectura e Interpretación de permisos]]
